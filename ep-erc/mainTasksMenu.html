About

  • An Tasks is any action that needs to be complete.
  • Each Task is listed in the table on the left - by clicking on a specific Task, its details will appear on the right.
  • To add a new Task, click on the Add Task Add Task link at the top of the Task list.
  • Mostly likely a User will only need to enter the task and hit enter - defaults are listed below
  • A task will default to the current user but can be change to assign the task to another user
  • A task will default to the current date for the open date but can be changed
  • A task will default to the the Open status but can be changed
  • To edit an Task, select the Task from the list on the left, click the Edit link will appear at the top of the Task Information section.
  • To delete a Task, select the Task from the list on the left, click the Delete link at the top of the Task Information section.
  • The Task Information section displays all of the details for the Task.
  • The Task Notes section enables you to enter in notes for this Task.
  • Only a User with Administrator role can delete a task.
